Project Overview:

This application is a website that is used for advertisements, and other things to attract customers to this freelance photography website. This website hosts examples pictures of photoshoots that would be offered. This website also hosts the pricings for different shoots.

  • Application purpose: To advertise and provide information about freelance photography services.
  • Intended users: Potential clients seeking photography services.
  • Website content overview:
    1. A portfolio gallery of photos.
    2. Pricing information for different photoshoots.
    3. Information about the photographer (About Me).
    4. Contact information and a contact form.

This website will have a photo gallery that will implemented with JavaScript. The photo gallery will be on the gallery page. The JavaScript will be used so that we can look at one image at a time and go to the next one when needed. In addition to the photo gallery there will also be a price calculator. The price calculator will be on the pricing page. JavaScript will be used so I can easily make useful calculator for the potential customers.
